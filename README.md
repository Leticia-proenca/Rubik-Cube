# Rubiks Cube 3x3 Simulator
An interactive Python - based Rubik's Cube simulator with visual representation and real-time manipulation. Perfect for learning cube notation, practicing algorithms, or just having fun!

✨ Features

🎨 Visual Representation: Real-time matplotlib visualization of the cube state
🎮 Interactive Controls: Execute moves using standard Rubik's Cube notation
🔀 Scramble Function: Randomly scramble the cube with customizable number of moves
📝 Move History: Track all moves performed on the cube
🔄 Reset Function: Instantly return to solved state
🎯 Multi-move Support: Execute multiple moves in sequence


For this you need to install the matplotlib and numpy library


🎮 How to Use
Basic Controls:
The simulator uses standard Rubik's Cube notation

Special Commands:
show - display current cube state
scramble - scramble the cube (default: 20 moves)
reset - reset cube to solved state
history - show aall moves performed
quit - exit the simulator
